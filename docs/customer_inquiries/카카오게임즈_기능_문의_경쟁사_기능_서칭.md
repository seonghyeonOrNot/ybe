## 1) 시간외근무 신청 ↔ 출퇴근 연동

| 항목 | Flex | 시프티 |
|------|------|--------|
| **연동 방식** | 초과근무 대시보드에서 실시간 확인 | 근무일정과 출퇴근기록 연동 |
| **승인 프로세스** | 근무 정책별 승인/알림 설정 가능 | 다단계 승인 규칙 설정 지원 (파트장+팀장 등) |

## 2) 선근무 후결재

| 항목 | Flex | 시프티 |
|------|------|--------|
| **자율 근무 기록** | ✅ 구성원이 자신의 근무시간 직접 기록 | ✅ 무일정 근무 출퇴근 기록 지원 |
| **후결재** | 요청자별 승인 정책 설정 가능 | 근무일정 수정 요청으로 사후 처리 |
| **미등록 처리** | 미입력 현황 대시보드에서 확인 | 무일정 기록 후 기존 일정과 연동 가능 |

**주요 차이점:**
- **Flex**: 자동근무기록 기능으로 자율적 근태 관리 가능, 위치 정보 미저장
- **시프티**: 무일정 근무 시 지각/결근 미반영, 후에 기존 일정과 연동 처리 가능

**Sources:**
- [Flex 근무 승인 설정](https://guide.flex.team/01a19757-9157-49a9-ae78-2109c19745a0)
- [시프티 무일정 근무](https://shiftee.io/ko/help/article/clockInOutOnCall)

## 3) 근로자별 출퇴근 기기 지정

| 항목 | Flex | 시프티 |
|------|------|--------|
| **기기 제한** | ❌ 개인별 기기 지정 미지원 | ✅ **1인 1기기 귀속** (모바일) |
| **제한 방식** | 근무지(GPS/IP) 단위로 제한 | 계정-기기 1:1 매핑으로 대리출퇴근 방지 |
| **PC 제한** | IP 기반 제한 가능 | PC-OFF 기능 (근무시간 외 PC 사용 제한) |
| **기기 변경** | - | 관리자에게 기기 변경 요청 필요 |

**주요 차이점:**
- **Flex**: GPS/IP 기반 **근무지 단위** 제한만 지원
- **시프티**: **개인별 모바일 기기 1:1 귀속** 기능 있음 ⭐

**Sources:**
- [Flex GPS/IP 제한](https://guide.flex.team/38d8e249-55d2-4d1d-8ac9-0aff3183fb79)
- [시프티 기기 초기화](https://shiftee.io/ko/help/article/resetEmployeeBoundDevice)
- [시프티 PC-OFF](https://shiftee.io/ko/shiftee-desktop)

## 4) 재택근무 GPS 기록 및 조회 권한

| 항목 | Flex | 시프티 |
|------|------|--------|
| **GPS 기록 저장** | ❌ **위치정보 미저장** (IN/OUT 결과만) | ❌ **위치정보 미저장** (인증 후 즉시 폐기) |
| **관리자 조회** | 근무지 내 여부만 확인 가능 | 반경 내 도착 여부만 확인 가능 |
| **권한별 조회** | 조회 권한 설정 가능 | 위치정보 자체를 저장하지 않음 |
| **재택근무** | 근무지 밖 등록 승인 기능 | 승인 후 출퇴근 가능 |

**주요 공통점:**
- **두 서비스 모두 개인정보 보호를 위해 GPS 상세 위치를 저장하지 않음**
- 근무지 반경 내 IN/OUT 여부만 확인

**Sources:**
- [Flex 근무지 GPS 설정](https://guide.flex.team/ko/articles/10283602-%EA%B7%BC%EB%AC%B4%EC%A7%80-gps-%EC%84%A4%EC%A0%95)
- [시프티 출퇴근 장소 좌표](https://shiftee.io/ko/help/article/timeclockAreaAndCoordinate)

## 시사점

### 기기 지정 기능
- 시프티가 개인별 모바일 기기 귀속 기능을 제공하고 있어, 카카오게임즈 요청사항 대응 시 참고할 수 있음
- 대리출퇴근 방지 목적으로 유용한 기능

### GPS 위치 조회
- 경쟁사 모두 개인정보 보호를 위해 상세 위치 미저장 정책을 채택
- 업계 표준으로 보이며, 고객에게 이 점을 설명하여 이해를 구할 수 있음

### 선근무 후결재
- Flex와 시프티 모두 유연한 근무 기록 방식을 지원
- 시프티의 "무일정 근무 후 기존 일정 연동" 기능은 YBE에서 참고할 만한 접근법
