## 1) 시간외근무 신청 시 출근 버튼 활성화 여부

### 문의 내용
> 추가 근무 신청(시간외근무신청)의 경우 조직장의 승인이 없어도 '출근'버튼이 활성화 되고 출근처리가 바로 되는건지?

### 답변

시간외근무 신청/승인과 출퇴근 등록은 **연동**됩니다.

| 구분 | 동작 방식 |
|------|----------|
| **평일** | 기본 근무 종료 후 → 시간외근무 진행 가능 |
| **휴일** | 시간외근무 신청/승인 완료 후 → 출퇴근 등록 가능 |

**⚠️ 유의사항 (휴일 시간외근무)**

실제 근무 시간이 신청 시간과 다를 경우:
- 현재 **휴일 근태 변경이 불가**합니다
- 기존 시간외근무 신청서 **취소 후 재상신** 필요
- (평일은 관리자가 PC 사용 현황 확인 후 시간 조정 가능)

## 2) 선근무 후결재 시 출퇴근 처리

### 문의 내용
> 시간외근무 신청을 하지 못했거나, 조직장 승인이 늦을 경우, 선근무, 후결재 시 출근처리는 어떻게 되는지? (실제 근무 시간이 지난 후 조직장의 승인이 후에 이루어져도 시간외근무가 정상 반영되는지)

### 답변

선근무 후결재 방식은 **기능적으로 가능**하나, 아래 제약사항을 안내드립니다.

| 항목 | 상태 |
|------|------|
| 시간외근무 문서 | ✅ 후결재로 승인 시 기록됨 |
| 출퇴근 근태 기록 | ❌ **미등록** (승인 전에는 출퇴근 등록 불가) |

**⚠️ 리스크 안내**

- 결재 승인이 실제 근무 시간 이후에 이루어지면 → **근태 데이터 없이 문서만 존재**
- 급여/근로시간 산정 시 **근태 기록 누락**으로 정합성 이슈 발생 가능
- **권장**: 시간외근무는 가급적 **사전 신청/승인 후 근무** 원칙 준수

## 3) 근로자별 출퇴근 기기 지정

### 문의 내용
> 근로자 개인별로 기기를 1개씩만 등록할 수 있도록 지정할 수 있는지? (예를 들어 A근로자는 모바일로만 출퇴근 등록할 수 있도록 지정, B근로자는 PC로만 가능하도록 지정)

### 답변

현재는 **근무지 단위**로 출퇴근 방식(GPS/비콘)을 설정하며, **구성원별 기기 지정 기능은 미지원**입니다.

| 현재 기능 | 요청 기능 |
|----------|----------|
| 근무지별 GPS/비콘 설정 | 구성원별 기기(모바일/PC) 지정 |
| ✅ 지원 | ❌ **추가 개발 필요** |

**개발 시 필요 항목:**
- 구성원별 출퇴근 방식 지정 기능
- 관리자 설정 화면 추가

## 4) 재택근무 GPS 기록 및 조회 권한

### 문의 내용
> 재택근무와 외근이 잦은 편인데, 재택근무 시 다중근무지로 근무지 지정해두면, GPS기록이 남을텐데 이 기록을 가장 상위의 최고관리자만 확인할 수 있도록 설정이 가능한가?

### 답변

| 항목 | 현재 상태 |
|------|----------|
| GPS 기록 | ✅ 기록됨 (근무지명 + 시간만 표시) |
| 상세 주소 노출 | ❌ **개인정보 보호로 비노출** |
| 권한별 조회 제한 | ❌ 현재 미지원 (추가 개발 필요) |
| 다중 거주지 등록 | ❌ 현재 1개만 등록 가능 (추가 개발 필요) |

**현재 동작:**
- 재택근무 시 GPS 좌표는 내부적으로 기록되나, **개인정보(상세 주소)는 화면에 노출되지 않음**
- 근태 현황에서는 **"재택"** 근무지명과 **시간**만 확인 가능

**추가 개발 필요 사항:**
1. 최고관리자만 GPS 상세 조회 가능하도록 **권한별 조회 제한** 기능
2. **다중 거주지 등록** 기능 (현재 1개 → 복수 등록)
