┌──────────┐        ┌──────────┐       ┌──────────┐        ┌──────────┐

    │                    │                  │                    │

    └── 문서 형태 설정   └── 대상자 수 확인                      ├── 엑셀 다운로드

#### 상세 플로우 설명

| 단계 | 화면 | 주요 액션 | 연결 화면 |
|------|------|----------|----------|
| 1. 촉진 설정 | 연차촉진 설정 페이지 | 소멸 기준일, 촉진 시작 시점, 제출 기한, 문서 형태(통합/개별) 설정 | 대상자 확인 |
| 2. 대상자 확인 | 대상자 리스트 페이지 | 촉진 대상자 필터링, 검색, 선택, 제외 처리 | 발송 설정 |
| 3. 발송 | 발송 설정 팝업 | 발송 시기 선택(즉시/예약), 알림 채널 선택, 발송 실행 | 모니터링 |
| 4. 모니터링 | 촉진 현황 대시보드 | 진행상태 확인, 필터/검색, 리마인드 발송, 엑셀 다운로드 | - |

### 1.2 사용자 신청/조회 플로우

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                           사용자 연차촉진 신청 플로우                              │
└─────────────────────────────────────────────────────────────────────────────────┘

[팝업 확인]          [계획서 작성]         [제출]              [일정변경]
    │                    │                  │                    │
    ▼                    ▼                  ▼                    ▼
┌──────────┐        ┌──────────┐       ┌──────────┐        ┌──────────┐
│ 촉진알림 │   →    │ 휴가일정 │   →   │ 제출확인 │   →    │ 일정     │
│ 팝업     │        │ 입력     │       │          │        │ 수정     │
└──────────┘        └──────────┘       └──────────┘        └──────────┘
    │                    │                  │                    │
    ├── 잔여연차 확인    ├── 날짜 선택      ├── 제출 완료        ├── 일정 수정
    ├── 제출 기한 확인   ├── 휴가 유형      │   (연차 미차감)    ├── 취소
    ├── 계획서 작성      ├── 임시저장       └── 결과 알림        └── 변경사유 입력
    └── 이전 계획서 조회 └── 자동저장                              (별도 승인 불필요)
```

#### 상세 플로우 설명

| 단계 | 화면 | 주요 액션 | 연결 화면 |
|------|------|----------|----------|
| 1. 팝업 확인 | 촉진 알림 팝업 | 잔여연차/이월연차 확인, 제출 기한 확인, 계획서 작성 진입 | 계획서 작성 |
| 2. 계획서 작성 | 계획서 작성 페이지 | 휴가 일정 입력, 휴가 유형 선택, 임시저장, 자동저장(30초) | 제출 확인 |
| 3. 제출 | 제출 확인 팝업 | 입력 내용 최종 확인, 제출 완료 (연차 차감 없음) | 제출 완료 |
| 4. 일정변경 | 계획서 수정 페이지 | 일정 변경/취소, 변경 사유 입력 (별도 승인 없이 자유 변경 가능) | 변경 완료 |

## 6. 결론

본 UX 설계 문서는 고객 VOC 52건(UI/UX 관련 9건 포함)을 기반으로 연차촉진 기능의 사용성을 대폭 개선하는 것을 목표로 합니다.

**핵심 개선 사항:**
1. **연차 차감 분리**: 계획서 제출과 연차 차감을 분리하여 사용자의 불편함 해소
2. **필터/검색 강화**: 100명 이상의 대상자도 효율적으로 관리 가능
3. **데이터 보호**: 임시저장/자동저장으로 작성 중 이탈 시 데이터 손실 방지
4. **UX 정상화**: 팝업 닫기 동작 정상화로 기본적인 사용성 확보
5. **정보 명확화**: 날짜 표기, 이월연차 표시 통일로 혼란 감소

이러한 개선을 통해 고객 문의를 크게 줄이고, 관리자와 사용자 모두의 업무 효율성을 높일 수 있을 것으로 기대됩니다.
