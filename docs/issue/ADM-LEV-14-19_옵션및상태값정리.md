# ADM-LEV-14~19 옵션값 & 상태값 정리

## 📋 개요
ADM-LEV-14부터 ADM-LEV-19까지의 기능에서 사용되는 **옵션값(Option Values)**과 **상태값(State Values)**을 구분하여 정리합니다.

---

## ADM-LEV-14: 기간 및 상세 필터

### 🔹 옵션값 (Option Values)
- **기간 선택**: 기간 범위 설정 (시작일~종료일)
- **입사일**: 필터링 조건
- **재직상태**: 필터링 조건
- **키워드 검색**: 구성원명 등 검색 조건

### 🔹 상태값 (State Values)
- 조회 데이터의 상태 (현황 반영)

---

## ADM-LEV-15: 조직도 필터

### 🔹 옵션값 (Option Values)
- **부서 선택**: 조직도 내 부서 단위 선택
- **개인 선택**: 조직도 내 개인 단위 선택

### 🔹 상태값 (State Values)
- 필터링 적용 상태

---

## ADM-LEV-16: 기본 정보 내역

### 🔹 옵션값 (Option Values)
- **부서명**: 표시 항목
- **구성원명**: 표시 항목
- **직위/직책**: 표시 항목
- **입사일**: 표시 항목
- **재직연수**: 표시 항목
- **연차 유형**: 표시 항목

### 🔹 상태값 (State Values)
- **상태**: 재직 상태 (활동/휴직/퇴직 등)

---

## ADM-LEV-17: 연차 관련 정보 내역

### 🔹 옵션값 (Option Values)
- **총 발생연차 (지급일수)**: 계산된 값
- **사용연차 (사용일수)**: 계산된 값
- **잔여연차 (잔여일수)**: 계산된 값
- **이월연차**: 이전 연도 이월분
- **조정연차 (조정일수)**: 관리자 조정분
- **소멸연차**: 사용하지 않아 소멸된 연차

### 🔹 상태값 (State Values)
- 연차 잔여 현황

---

## ADM-LEV-18: 관리 > 조정 버튼

### 🔹 옵션값 (Option Values)
- **조정 방식**: 일괄 조정 / 개별 조정
- **조정 방향**: 차감 / 증감

### 🔹 상태값 (State Values)
- **조정 처리 상태**: 대기중 / 처리완료 / 취소

---

## 📊 요약 정리

### 전체 상태값 목록
```
[결재/처리 상태]
- 확정 / 취소&반려 / 대기중
- 처리됨 / 미처리 / 취소

[재직 상태]
- 활동 / 휴직 / 퇴직 등

[연차 분류]
- 1년 미만 연차
- 비례 연차
- 대상 휴가
- 비대상 휴가
- 기본 연차
- 이월 연차
```

### 전체 옵션값 목록
```
[필터 옵션]
- 기간 선택
- 입사일 필터
- 재직상태 필터
- 키워드 검색
- 부서 선택
- 개인 선택

[조정 옵션]
- 조정 방식: 일괄 / 개별
- 조정 방향: 차감 / 증감

[표시 항목]
- 부서명, 구성원명, 직위/직책
- 입사일, 재직연수, 연차 유형
- 연차 관련 수치 (발생, 사용, 잔여, 소멸 등)

[사용 단위]
- 일 / 시간
```
