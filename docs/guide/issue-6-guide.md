## 1. 기능 배경 (Background)

### 고객/운영 측 불편

기존에는 출장, 외근, 재택근무 등의 근무 형태 변경 시 종이 문서나 이메일을 통해 승인을 요청해야 했으며, 이로 인해 다음과 같은 문제가 발생했습니다.

- 승인 절차가 지연되어 긴급한 일정 조정이 어려움
- 현재 구성원의 근무 위치를 실시간으로 파악하기 어려움
- 출장이나 외근 시 별도로 위치 정보를 공유해야 하는 번거로움
- 과거 근무 이력 추적 및 관리의 어려움

### 기존 방식의 한계

- 종이 결재 방식: 물리적 이동과 시간 소요, 분실 위험
- 이메일 방식: 승인 여부 추적 어려움, 일정 자동 반영 불가
- 수동 관리: 근무지 변경 사항의 실시간 반영 불가, 데이터 집계 어려움

### 이번 기능이 해결하는 핵심 문제

전자결재 시스템을 통해 출장, 외근, 재택근무 신청부터 승인, 일정 등록, 근무 시작/종료까지 전 과정을 디지털화하여 효율적인 근무 관리를 가능하게 합니다.

## 2. 기능 요약 (Summary)

전자결재 시스템은 출장, 외근, 재택근무 등 다양한 근무 형태에 대한 신청과 승인을 온라인으로 처리할 수 있는 통합 시스템입니다. 사용자는 웹과 모바일 앱을 통해 언제 어디서나 결재 신청 및 승인을 수행할 수 있으며, 승인된 일정은 자동으로 캘린더에 등록되어 편리하게 관리됩니다.

### 핵심 포인트

- **통합 결재 프로세스**: 출장, 외근, 재택, 파견, 시간외근무 등 모든 근무 형태를 하나의 시스템에서 신청 및 관리
- **실시간 근무 현황 파악**: 모바일 앱에서 오늘의 출장/외근 일정을 한눈에 확인하고, 근무지 상태 배지를 통해 현재 위치를 실시간으로 표시
- **자동화된 일정 관리**: 승인된 전자결재는 자동으로 사용자 캘린더에 등록되며, 푸시 알림을 통해 일정 시작 전 미리 알림 제공

## 3. 기능 상세 설명 (Detailed Description)

### 웹 기안서 작성 흐름

1. **신청서 선택**: 사용자는 웹 시스템에서 출장신청서, 재택근무신청서, 외근신청서, 파견신청서, 시간외근무신청서 중 필요한 양식을 선택합니다.

2. **기본 정보 입력**:
   - 제목(필수): 신청 내용을 요약한 제목 입력
   - 신청일: 캘린더 피커를 통해 시작일과 종료일 선택 (최대 7일까지 연속 선택 가능)
   - 당일 체크박스: 시작일과 종료일이 동일한 경우 체크

3. **시간 및 근무지 설정**:
   - 시작/종료 시간을 시와 분 단위로 입력
   - 근무지 구분(선택): 근무 장소 유형을 선택
   - '+ 추가' 버튼: 동일 날짜에 여러 시간대 등록 가능
   - '종일일정' 체크: 체크 시 시간 선택 비활성화

4. **개별 설정 (선택사항)**:
   - '개별 설정' 토글 활성화 시, 선택한 기간의 각 날짜별로 독립적인 시간/근무지 설정 가능
   - '전체 반영' 버튼: 특정 일자의 설정을 전체 날짜에 적용
   - '초기화' 버튼: 입력된 모든 개별 데이터를 삭제

5. **휴게 시간 자동 계산**:
   - 근로기준법에 따라 휴게 시간이 자동으로 계산되어 표시됨
   - 4시간 근무: 30분 휴게
   - 8시간 근무: 1시간 휴게

6. **상세 내용 작성**:
   - 리치 텍스트 에디터를 통해 출장 목적, 계획 등을 자유롭게 작성
   - 하단에 현재 입력된 단어 수 표시

7. **첨부 파일 및 관련 문서**:
   - 드래그 앤 드롭 방식으로 파일 첨부
   - 관련 결재 문서를 검색하여 연결 가능

8. **결재 처리**:
   - 결재라인 지정: 승인자 선택
   - 임시저장: 3분 간격 자동저장 또는 수동 저장
   - 미리보기: 최종 제출 전 내용 확인
   - 결재요청: 필수값이 모두 입력되면 버튼 활성화, 클릭 시 결재선 지정자에게 푸시 알림 발송

### 모바일 일정 확인 및 근무 시작 흐름

1. **오늘의 일정 확인**:
   - 모바일 앱 메인 화면에서 오늘 예정된 출장/외근 일정의 총 건수를 상단에 표시
   - 시간 순서대로 일정을 나열하며, 현재 시각 기준으로 상태 구분
     - 진행중: 파란색 배지(ing)
     - 기본: 일반 표시
     - 종료: 회색 처리

2. **일정 상세 확인**:
   - 각 일정 항목 클릭 시 상세 내용 확인 가능
   - X 버튼 클릭 시 해당 일정을 리스트에서 숨김 처리
   - 3개 초과 시 페이징 처리

3. **근무 시작**:
   - 근무 시작 버튼 클릭 시 근무지별(재택/파견/출장/외근) 팝업 표시
   - 위치 정보와 메모 입력 칸 제공
   - 메모: 최대 30자까지 입력 가능 (선택사항)
   - '바로 시작' 클릭 시 출근 처리 완료

4. **근무 메모 관리**:
   - 입력한 메모는 일정 상세 모달의 메모 내역 탭에서 조회/수정/삭제 가능
   - '상세 내역보기 >' 클릭 시 과거 메모 이력 확인 페이지로 이동

5. **근무지 상태 배지**:
   - 메인 화면 우측 상단에 현재 근무 장소에 따라 배지 표시
   - 상태: 사무실, 재택, 파견, 출장, 미출근, 퇴근, 외근, 이동중

## 4. 대상 사용자 / 권한

### 사용자 (일반 구성원)

**권한**:
- 본인의 전자결재 신청서 작성 및 제출
- 본인의 결재 현황 조회 (상신한 문서)

**접근 위치**:
- 웹: 전자결재 > 기안서 작성 > [출장/재택/외근/파견/시간외근무]신청서

**권한**:
- 결재선에 지정된 경우 해당 결재 문서 승인/반려
- 결재할 문서 목록 조회

**접근 위치**:
- 웹: 전자결재 > 결재 현황 > 결재할문서

**권한**:

- 결재 정책 설정 (최대 신청 일수, 자동저장 주기 등)

## 5. 사용 방법(단계별)

### 웹에서 출장신청서 작성 및 제출

**Step 1**: 웹 시스템에 로그인한 후, 좌측 메뉴에서 "전자결재" > "기안서 작성" > "출장신청서"를 선택합니다.

**Step 2**: 제목 입력란에 출장 목적을 간단히 입력합니다. (예: "부산 지사 방문")

**Step 3**: 신청일 선택 영역에서 캘린더 아이콘을 클릭하고, 출장 시작일과 종료일을 선택합니다. 당일 출장인 경우 "당일" 체크박스를 선택합니다.

**Step 4**: 신청 시간 영역에서 시작 시간과 종료 시간을 시/분 단위로 입력하고, 필요 시 "근무지 구분"을 선택합니다. 동일 날짜에 여러 시간대가 있는 경우 "+ 추가" 버튼을 클릭하여 추가 시간대를 입력합니다.

**Step 5**: 여러 날짜에 걸친 출장이며 날짜별로 시간이 다른 경우, "개별 설정" 토글을 활성화하여 각 날짜별로 시작/종료 시간을 개별 설정합니다.

**Step 6**: 휴게 시간이 자동 계산되어 표시되는지 확인하고, 리치 텍스트 에디터에 출장 목적과 상세 계획을 작성합니다.

**Step 7**: 필요한 경우 첨부 파일을 드래그 앤 드롭으로 업로드하고, 관련 문서가 있다면 검색하여 연결합니다.

**Step 8**: 우측 상단의 "결재라인 지정" 버튼을 클릭하여 승인자를 선택합니다.

**Step 9**: 모든 내용을 확인한 후 "결재요청" 버튼을 클릭하여 제출합니다. (필수값이 누락된 경우 버튼이 비활성화됩니다)

**Step 10**: 결재가 승인되면 푸시 알림을 받게 되며, 승인된 일정은 자동으로 캘린더에 등록됩니다.

### 모바일에서 오늘의 외근 일정 확인 및 근무 시작

**Step 1**: 모바일 앱을 실행하면 메인 화면 상단에 오늘의 출장/외근 일정 건수가 표시됩니다.

**Step 2**: 일정 리스트에서 오늘 예정된 외근 일정을 확인합니다. 시간 순서대로 나열되며, 현재 진행 중인 일정은 파란색 "ing" 배지가 표시됩니다.

**Step 3**: 해당 일정 항목을 클릭하여 상세 내용(시간, 장소, 목적 등)을 확인합니다.

**Step 4**: 일정 시작 시간이 되면, "근무 시작" 버튼을 클릭합니다.

**Step 5**: 팝업이 나타나면 위치 정보가 자동으로 표시되고, 필요 시 메모 입력란에 최대 30자까지 메모를 작성합니다. (예: "거래처 도착")

**Step 6**: "바로 시작" 버튼을 클릭하여 출근 처리를 완료합니다.

**Step 7**: 메인 화면 우측 상단의 배지가 "외근"으로 변경되었는지 확인합니다.

**Step 8**: 일정 중 추가로 메모를 남기고 싶다면, 일정 항목을 다시 클릭하여 상세 모달의 "메모 내역" 탭에서 메모를 조회/수정/삭제할 수 있습니다.

## 7. 주의사항/제한

### 정책 제약

1. **신청 기간 제한**: 출장, 재택, 외근, 파견 신청 시 최대 7일까지 연속 선택 가능합니다. 7일을 초과하는 경우 별도의 신청서를 추가로 제출해야 합니다.

2. **제목 필수 입력**: 모든 신청서에서 제목은 필수 입력 항목입니다. 제목이 입력되지 않으면 결재요청 버튼이 비활성화되어 제출할 수 없습니다.

3. **근무지 구분 미선택 시 경고**: 근무지 구분은 선택 항목이지만, 미선택 시 경고 메시지가 표시될 수 있습니다. 정확한 근무지 기록을 위해 가능한 한 선택을 권장합니다.

4. **종료 시간 유효성**: 시간외근무신청서 작성 시 종료 시간은 시작 시간보다 빠를 수 없습니다. 유효하지 않은 시간 입력 시 오류 메시지가 표시됩니다.

5. **메모 길이 제한**: 모바일 앱에서 근무 시작 시 입력하는 메모는 최대 30자까지 제한됩니다. 30자를 초과하면 입력이 차단됩니다.

### 데이터 반영 시점

1. **자동저장**: 웹 기안서 작성 중 3분 간격으로 자동저장이 실행됩니다. 브라우저를 종료하더라도 자동저장된 시점까지의 내용은 임시저장 목록에서 불러올 수 있습니다.

2. **캘린더 등록**: 결재가 승인되면 즉시 사용자 캘린더에 일정이 자동 등록됩니다. 단, 결재가 반려되거나 철회된 경우 캘린더에 등록되지 않습니다.

3. **푸시 알림 발송**: 결재 상신 시 결재선 지정자에게 즉시 푸시 알림이 발송됩니다. 일정 시작 10분 전에도 사용자에게 알림이 발송됩니다.

4. **근무지 배지 갱신**: 모바일 앱에서 근무 시작 버튼을 클릭하고 "바로 시작"을 누르면, 메인 화면의 근무지 상태 배지가 즉시 갱신됩니다.

### 자주 발생하는 오해

1. **"당일 체크 시 시간이 사라져요"**: '당일' 체크박스는 시작일과 종료일을 동일하게 설정하는 기능일 뿐, 시간 입력과는 무관합니다. 시간은 별도로 입력해야 합니다. '종일일정' 체크박스를 선택하면 시간 선택이 비활성화됩니다.

2. **"개별 설정 후 초기화하면 전체가 삭제되나요?"**: 네, '초기화' 버튼은 개별 설정 모드에서 입력된 모든 날짜의 데이터를 삭제합니다. 신중하게 사용해야 합니다.

3. **"휴게 시간을 직접 수정할 수 있나요?"**: 휴게 시간은 근로기준법에 따라 자동 계산되며, 사용자가 직접 수정할 수 없습니다. 4시간 근무 시 30분, 8시간 근무 시 1시간이 자동 적용됩니다.

4. **"모바일에서 X 버튼을 누르면 일정이 삭제되나요?"**: 아니요, X 버튼은 해당 일정을 리스트에서 숨기는 기능일 뿐, 일정 자체가 삭제되지는 않습니다. 일정 상세 페이지나 캘린더에서는 여전히 확인할 수 있습니다.

5. **"일정이 3개 이상이면 어떻게 보나요?"**: 모바일 앱은 일정이 3개를 초과할 경우 자동으로 페이징 처리됩니다. 화면을 위아래로 스크롤하여 모든 일정을 확인할 수 있습니다.

6. **"결재라인을 지정하지 않고 제출할 수 있나요?"**: 아니요, 결재라인 지정은 필수입니다. 결재라인이 지정되지 않으면 결재요청 버튼이 비활성화되어 제출할 수 없습니다.

## 8. FAQ (최소 7개)

### Q1. 출장신청서를 제출했는데 수정하고 싶어요. 어떻게 하나요?

**A**: 결재가 완료되기 전이라면 "전자결재 > 나의 현황"에서 해당 문서를 선택하여 철회할 수 있습니다. 철회 후 다시 기안서를 작성하여 제출하세요. 결재가 완료된 후에는 기존 신청서를 취소하고 새로 신청해야 합니다.

### Q2. 신청 기간을 7일 이상 선택할 수 없는데, 10일 출장은 어떻게 신청하나요?

**A**: 출장, 재택, 외근 등의 신청은 최대 7일까지 한 번에 선택 가능합니다. 10일 출장의 경우, 첫 7일을 먼저 신청하고, 나머지 3일을 별도의 신청서로 추가 제출해야 합니다.

### Q3. 모바일에서 근무 시작 버튼을 눌렀는데 메모를 안 썼어요. 나중에 추가할 수 있나요?

**A**: 네, 가능합니다. 메인 화면에서 해당 일정 항목을 클릭하여 상세 모달을 열고, "메모 내역" 탭에서 메모를 추가하거나 수정할 수 있습니다.

### Q4. 휴게 시간이 자동 계산되는데, 실제로는 1시간 30분 쉬었어요. 수정할 수 있나요?

**A**: 휴게 시간은 근로기준법에 따라 자동 계산되며 사용자가 직접 수정할 수 없습니다. 실제 휴게 시간이 다른 경우, 상세 내용 작성란에 특이사항으로 기재하거나 관리자에게 별도로 문의하세요.

### Q5. 결재가 승인되면 자동으로 캘린더에 등록되나요?

**A**: 네, 결재가 최종 승인되면 신청한 일정이 자동으로 사용자 캘린더(USR-SCH)에 등록됩니다. 별도로 캘린더에 일정을 입력할 필요가 없습니다.

### Q6. 모바일 메인 화면 상단의 배지가 "미출근"으로 표시되는데, 정상인가요?

**A**: 네, 정상입니다. 근무지 상태 배지는 현재 근무 상태를 실시간으로 표시합니다. 출근 전에는 "미출근", 근무 시작 후에는 "사무실", "재택", "출장", "외근" 등으로 변경됩니다. 퇴근 처리 후에는 "퇴근"으로 표시됩니다.

### Q7. '개별 설정' 모드에서 '전체 반영' 버튼을 누르면 어떻게 되나요?

**A**: '전체 반영' 버튼을 클릭하면 현재 설정된 특정 날짜의 시간 및 근무지 정보가 선택한 전체 기간의 모든 날짜에 동일하게 적용됩니다. 예를 들어, 2월 10일의 시간을 09:00~18:00으로 설정하고 '전체 반영'을 누르면, 2월 11일, 12일 등 전체 기간이 모두 09:00~18:00으로 설정됩니다.

### Q8. 결재요청 버튼이 비활성화되어 있어요. 왜 그런가요?

**A**: 필수 입력 항목이 누락되었을 가능성이 높습니다. 다음 항목들을 확인하세요: (1) 제목 입력 여부, (2) 신청일 선택 여부, (3) 결재라인 지정 여부. 모든 필수 항목이 입력되면 버튼이 활성화됩니다.

### Q9. 시간외근무신청서에서 "휴일" 체크박스는 언제 사용하나요?

**A**: 시간외근무신청서에서 휴일(토요일, 일요일, 공휴일) 근무를 신청할 경우 "휴일" 체크박스를 활성화하세요. 이를 통해 휴일 근무 시간을 별도로 집계하며, 근무 수당 계산 시 반영될 수 있습니다.

### Q10. 모바일에서 일정 항목의 X 버튼을 눌렀는데, 다시 보고 싶어요. 어떻게 하나요?

**A**: X 버튼은 해당 일정을 리스트에서 일시적으로 숨기는 기능입니다. 앱을 재시작하거나 일정 상세 페이지에서 다시 확인할 수 있습니다. 일정 자체가 삭제되는 것은 아니므로 걱정하지 않으셔도 됩니다.
