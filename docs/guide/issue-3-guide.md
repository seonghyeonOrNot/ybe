# 연차/휴가 시스템 이용 가이드

## 1. 개요

본 가이드는 근로기준법에 따른 연차 및 휴가 관리 시스템의 이용 방법을 안내합니다. 이 시스템은 직원의 연차 발생, 사용, 소멸 및 대체 휴가 관리를 자동화하여 효율적인 휴가 관리를 지원합니다.

**주요 기능:**
- 자동 연차 발생 및 부여
- 연차 사용 신청 및 승인
- 연차 소멸 알림
- 대체 휴가 관리
- 연차 이월 처리

**법적 근거:** 근로기준법 제60조, 제61조

## 3. 사용 방법 (단계별)

### 3.1 연차 잔여 현황 조회

1. 시스템에 로그인합니다.
2. 메인 대시보드에서 "연차 관리" 메뉴를 선택합니다.
3. "나의 연차 현황"에서 다음 정보를 확인합니다:
   - 총 연차 일수
   - 사용한 연차
   - 남은 연차
   - 올해 소멸 예정 연차
   - 연차 사용 내역

### 3.2 연차 사용 신청

1. "연차 관리" > "연차 신청" 메뉴를 선택합니다.
2. 연차 사용 정보를 입력합니다:
   - **시작일**: 연차 사용 시작 날짜
   - **종료일**: 연차 사용 종료 날짜
   - **사용 유형**: 전일 (8시간) / 반차 (4시간) / 시간 단위
   - **사유**: 연차 사용 사유 (선택)
3. "신청" 버튼을 클릭합니다.
4. 승인 대기 상태로 전환됩니다.
5. 관리자 승인 후 연차가 차감됩니다.

**주의사항:**
- 최소 사용 단위는 0.5일입니다.
- 연차는 발생일이 오래된 것부터 자동으로 사용됩니다 (FIFO).
- 소멸 예정 연차가 우선 사용됩니다.

### 3.3 대체 휴가 신청

1. "연차 관리" > "대체 휴가 신청" 메뉴를 선택합니다.
2. 연장/휴일 근무 내역을 선택합니다.
3. 대체 휴가 사용 날짜를 선택합니다.
4. "신청" 버튼을 클릭합니다.

**대체 휴가 규정:**
- 연장/휴일 근무 시간의 1.5배가 대체 휴가로 부여됩니다.
- 발생일로부터 3개월 이내에 사용해야 합니다.
- 미사용 시 자동 소멸됩니다.

### 3.4 연차 승인 (관리자)

1. "연차 관리" > "승인 대기 목록" 메뉴를 선택합니다.
2. 팀원의 연차 신청 내역을 확인합니다:
   - 신청자 정보
   - 연차 사용 기간
   - 잔여 연차 일수
3. 승인 또는 반려를 선택합니다.
4. 반려 시 사유를 입력합니다.

### 3.5 연차 소멸 알림 확인

시스템은 연차 소멸 예정일 기준으로 자동 알림을 발송합니다:
- **90일 전**: 소멸 예정 연차 1차 알림
- **60일 전**: 소멸 예정 연차 2차 알림
- **30일 전**: 소멸 예정 연차 3차 알림
- **7일 전**: 소멸 예정 연차 최종 알림

알림 확인 방법:
1. 시스템 알림 센터에서 확인
2. 이메일로 수신
3. 대시보드 상단 알림 배너

## 5. 주의사항/제한

### 연차 사용 제한
1. **최소 신청 단위**: 0.5일 (4시간)
2. **신청 기한**: 최소 1일 전까지 신청 (긴급 상황 제외)
3. **동시 사용 제한**: 팀 내 30% 이상 동시 연차 사용 불가 (부서별 정책에 따라 상이)
4. **연차 사용 순서**: 발생일이 오래된 연차부터 자동 차감 (FIFO)

### 연차 소멸 규정
1. **소멸 기한**: 연차 발생일로부터 2년
2. **자동 소멸**: 기한 경과 시 자동 소멸 (복구 불가)
3. **소멸 예외**: 회사 귀책사유 또는 장기 휴직 시 예외 적용 가능

### 대체 휴가 제한
1. **발생 조건**: 연장 근무 또는 휴일 근무 시 발생
2. **사용 기한**: 발생일로부터 3개월
3. **보상 비율**: 근무 시간의 1.5배
4. **우선순위**: 대체 휴가가 연차보다 우선 사용됨

### 시스템 사용 제한
1. **승인 권한**: 직속 상급자만 승인 가능
2. **취소 제한**: 승인 완료 후 취소는 관리자 승인 필요
3. **소급 신청**: 과거 날짜 연차 신청 시 인사 담당자 승인 필요

## 6. FAQ (자주 묻는 질문)

### Q1. 입사 첫 해에는 연차가 몇 일 발생하나요?
**A:** 입사 첫 해에는 매월 1일의 연차가 발생하며, 최대 11일까지 발생합니다.
- 1개월 근무: 1일
- 2개월 근무: 2일
- ...
- 11개월 근무: 11일

1년 만근 후에는 15일의 연차가 추가로 발생합니다.

### Q2. 연차를 사용하지 않으면 어떻게 되나요?
**A:** 연차는 발생일로부터 2년 후 자동 소멸됩니다. 시스템은 소멸 예정일 90일, 60일, 30일, 7일 전에 알림을 발송하여 미리 사용할 수 있도록 안내합니다. 소멸된 연차는 복구할 수 없으므로 기한 내 사용을 권장합니다.

### Q3. 반차는 어떻게 사용하나요?
**A:** 연차 신청 시 "사용 유형"에서 "반차 (4시간)"를 선택하면 됩니다. 반차는 오전 반차 또는 오후 반차로 사용 가능하며, 0.5일이 차감됩니다. 최소 사용 단위는 0.5일입니다.

### Q4. 대체 휴가는 언제 발생하나요?
**A:** 다음의 경우 대체 휴가가 발생합니다:
- 법정 근로시간을 초과하여 연장 근무한 경우
- 휴일에 근무한 경우

대체 휴가는 근무 시간의 1.5배로 부여되며, 발생일로부터 3개월 이내에 사용해야 합니다. 예를 들어, 8시간 연장 근무 시 12시간(1.5일)의 대체 휴가가 부여됩니다.

### Q5. 연차 신청 후 취소할 수 있나요?
**A:**
- **승인 대기 상태**: 직원이 직접 취소 가능
- **승인 완료 상태**: 관리자에게 취소 요청 필요
- **사용 완료 상태**: 취소 불가

연차 사용일 전날까지는 취소가 가능하나, 당일 취소는 긴급 상황에 한해 관리자 승인 후 가능합니다.

### Q6. 연차를 시간 단위로 사용할 수 있나요?
**A:** 네, 시간 단위 연차 사용이 가능합니다. 연차 신청 시 "사용 유형"에서 "시간 단위"를 선택하고 사용할 시간을 입력하면 됩니다. 단, 최소 사용 단위는 0.5일(4시간)입니다.
- 1시간 사용: 0.125일 차감
- 2시간 사용: 0.25일 차감
- 4시간 사용: 0.5일 차감

### Q7. 연차가 자동으로 이월되나요?
**A:** 네, 사용하지 않은 연차는 자동으로 다음 해로 이월됩니다. 단, 이월된 연차는 발생일로부터 2년 후 자동 소멸되므로, 소멸 예정 연차를 우선적으로 사용하는 것이 좋습니다. 시스템은 FIFO(First-In-First-Out) 방식으로 가장 오래된 연차부터 자동 차감합니다.

### Q8. 퇴사 시 남은 연차는 어떻게 처리되나요?
**A:** 퇴사 시 남은 연차는 근로기준법에 따라 금전으로 보상받을 수 있습니다. 인사 담당자가 퇴사 처리 시 자동으로 정산되며, 최종 급여에 포함되어 지급됩니다. 연차 수당은 통상임금을 기준으로 계산됩니다.
