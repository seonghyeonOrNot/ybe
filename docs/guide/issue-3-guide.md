# 연차/휴가 시스템 이용 가이드

## 1. 기능 배경 (Background)

### 고객/운영 측 불편
기존 연차 관리는 대부분 수작업으로 진행되었으며, 엑셀이나 종이 문서를 통해 연차 신청과 승인이 이루어졌습니다. 이로 인해 다음과 같은 불편이 있었습니다:
- 연차 잔여 일수를 실시간으로 확인하기 어려움
- 연차 소멸 시점을 놓쳐 권리를 상실하는 경우 발생
- 관리자의 승인 처리가 지연되거나 누락되는 경우 발생
- 대체 휴가 발생 및 사용 내역 추적의 어려움
- 근로기준법 준수 여부를 확인하기 어려움

### 기존 방식의 한계
- **가시성 부족**: 개인별 연차 발생, 사용, 잔여 현황을 한눈에 파악하기 어려움
- **정확성 문제**: 수기 입력으로 인한 오류 발생 가능성
- **추적 불가능**: 과거 연차 사용 이력 및 승인 내역 조회 어려움
- **알림 부재**: 연차 소멸 임박 시 자동 알림 기능 없음
- **법적 리스크**: 근로기준법 미준수로 인한 법적 분쟁 가능성

### 이번 기능이 해결하는 핵심 문제
본 시스템은 근로기준법에 기반한 연차 자동 생성, 실시간 잔여 현황 조회, 연차 소멸 알림, 대체 휴가 자동 계산 등을 통해 연차 관리의 투명성과 정확성을 확보하고, 직원과 관리자 모두의 업무 효율을 높이는 것을 목표로 합니다.

---

## 2. 기능 요약 (Summary)

본 시스템을 통해 직원과 관리자는 연차 및 휴가를 체계적으로 관리할 수 있습니다. 근로기준법에 따라 연차가 자동으로 생성되며, 실시간으로 잔여 현황을 확인하고, 소멸 전 알림을 받을 수 있습니다.

**핵심 포인트 3가지:**
- **자동화된 연차 생성**: 입사일, 근무 형태에 따라 연차가 자동으로 생성되고 부여됨
- **실시간 현황 조회**: 언제든지 나의 연차 잔여 일수, 사용 내역, 소멸 예정 일수를 확인 가능
- **소멸 방지 알림**: 연차 소멸 90일, 60일, 30일, 7일 전에 자동 알림 발송으로 권리 보호

## 3. 기능 상세 설명 (Detailed Description)

### 연차 생성 및 부여 흐름

#### 입사 첫 해 (1년 미만 근무자)
1. 매월 1일에 자동으로 1일의 연차가 생성됩니다 (최대 11일)
2. 생성된 연차는 즉시 사용 가능한 상태로 부여됩니다
3. 예시:
   - 2025년 3월 15일 입사 → 4월 1일에 1일 생성, 5월 1일에 1일 생성...
   - 2026년 2월까지 총 11일의 연차 보유

#### 입사 2년차 이후
1. 매년 입사 기념일에 연차가 일괄 생성됩니다
2. 기본 15일 + 근속 연수에 따른 가산 (2년마다 1일 추가, 최대 25일)
3. 예시:
   - 3년차: 15일
   - 5년차: 16일 (15 + 1)
   - 7년차: 17일 (15 + 2)
   - 21년차 이상: 25일 (최대치)

### 연차 사용 흐름
1. **신청**: 직원이 시스템에서 연차 사용일과 단위(전일/반일/시간)를 선택하여 신청
2. **검증**: 시스템이 잔여 연차 확인 및 중복 신청 여부 자동 검증
3. **승인**: 관리자에게 승인 요청 알림 발송 → 관리자 승인/반려 처리
4. **차감**: 승인 완료 시 잔여 연차에서 자동 차감 (선입선출 원칙 적용)
5. **기록**: 모든 신청 및 승인 이력이 시스템에 자동 기록

### 연차 소멸 및 이월 흐름
1. **자동 이월**: 매년 미사용 연차는 자동으로 다음 연도로 이월됩니다
2. **소멸 알림**: 발생일로부터 2년이 경과하면 소멸되며, 소멸 90일/60일/30일/7일 전에 알림 발송
3. **우선 사용**: 연차 사용 시 소멸 임박 순으로 자동 차감 (FIFO)
4. **예외 케이스**: 회사의 귀책 사유로 사용하지 못한 경우, 별도 연장 절차 가능

### 대체 휴가 흐름
1. **발생**: 휴일 근무 시 1.5배의 대체 휴가 자동 생성 (예: 8시간 근무 → 12시간 대체 휴가)
2. **사용 기한**: 발생일로부터 3개월 이내 사용 필요
3. **우선순위**: 대체 휴가가 일반 연차보다 우선 사용됨
4. **소멸**: 3개월 경과 시 자동 소멸 (소멸 30일/7일 전 알림 발송)

### 주요 상태 및 조건
- **대기**: 신청 완료, 관리자 승인 대기 중
- **승인**: 관리자 승인 완료, 연차 사용 확정
- **반려**: 관리자 반려, 연차 차감되지 않음
- **취소**: 직원이 승인 전 신청 취소 가능 (승인 후 취소는 관리자 승인 필요)

### 예시 시나리오

**예시 1: 입사 첫 해 직원의 연차 사용**
- 입력: 2025년 3월 15일 입사, 8월 10일에 반차 신청
- 결과:
  - 4월~8월까지 5일 연차 보유
  - 반차 신청 시 0.5일 차감
  - 잔여 4.5일

**예시 2: 5년차 직원의 연차 소멸 알림**
- 입력: 2020년 1월 1일 입사, 2023년 연차 5일 미사용
- 결과:
  - 2024년 10월 3일: 90일 전 알림 (5일 소멸 예정)
  - 2024년 11월 2일: 60일 전 알림
  - 2024년 12월 2일: 30일 전 알림
  - 2024년 12월 25일: 7일 전 알림
  - 2025년 1월 1일: 5일 자동 소멸

## 4. 대상 사용자 / 권한

### 일반 직원
- **접근 위치**: 본인 계정으로 로그인 후 "내 연차" 메뉴
- **가능한 작업**:
  - 본인의 연차 잔여 현황 조회
  - 연차 사용 내역 조회
  - 연차 신청 및 취소
  - 대체 휴가 현황 조회

### 관리자 (팀장, 부서장 등)
- **접근 위치**: 관리자 계정으로 로그인 후 "근태관리 > 연차관리" 메뉴
- **가능한 작업**:
  - 소속 직원의 연차 신청 승인/반려
  - 소속 직원의 연차 현황 조회
  - 연차 사용 통계 확인
  - 연차 대량 조회 및 내보내기

### 인사 담당자 (HR)
- **접근 위치**: 관리자 계정으로 로그인 후 "인사관리 > 연차설정" 메뉴
- **가능한 작업**:
  - 전사 직원의 연차 현황 조회
  - 연차 정책 설정 (소수점 처리, 사용 단위 등)
  - 수동 연차 조정 (특별 휴가, 예외 케이스)
  - 대체 휴가 발생 등록
  - 연차 통계 및 리포트 생성

### 접근 조건
- 회사 계정으로 로그인 필요
- 권한별로 접근 가능한 메뉴가 자동 제어됨
- 개인정보 보호를 위해 타인의 연차 상세 내역은 권한이 있는 사용자만 조회 가능

## 5. 사용 방법 (단계별)

### 직원: 연차 신청하기

**Step 1: 로그인 및 메뉴 접근**
- 회사 계정으로 시스템에 로그인합니다
- 상단 메뉴에서 "내 연차" 또는 "근태관리"를 클릭합니다

**Step 2: 연차 신청 화면 진입**
- "연차 신청" 버튼을 클릭합니다
- 현재 잔여 연차 일수가 화면 상단에 표시됩니다

**Step 3: 신청 정보 입력**
- 연차 사용 시작일과 종료일을 선택합니다 (연속된 날짜 선택 가능)
- 사용 단위를 선택합니다: 전일(8시간) / 반일(4시간) / 시간 단위
- 사용 사유를 간단히 입력합니다 (선택 사항)

**Step 4: 신청 내용 확인**
- 입력한 정보를 확인합니다
- 차감될 연차 일수가 자동 계산되어 표시됩니다
- "신청하기" 버튼을 클릭합니다

**Step 5: 승인 대기 및 확인**
- 신청이 완료되면 관리자에게 승인 요청이 전송됩니다
- "내 신청 내역"에서 승인 상태를 실시간으로 확인할 수 있습니다
- 승인/반려 시 알림이 발송됩니다

### 관리자: 연차 승인하기

**Step 1: 승인 요청 알림 확인**
- 연차 신청이 접수되면 이메일 또는 시스템 알림을 받습니다
- 상단 알림 아이콘에 미처리 건수가 표시됩니다

**Step 2: 승인 대기 목록 조회**
- "근태관리 > 연차관리 > 승인 대기" 메뉴로 이동합니다
- 소속 직원의 대기 중인 신청 목록이 표시됩니다

**Step 3: 신청 내용 검토**
- 신청 건을 클릭하여 상세 내용을 확인합니다
- 신청자의 현재 연차 잔여 일수, 과거 사용 내역을 확인할 수 있습니다
- 팀 내 동일 기간 연차 사용 현황을 확인할 수 있습니다

**Step 4: 승인 또는 반려 처리**
- 승인 시: "승인" 버튼을 클릭합니다
- 반려 시: "반려" 버튼을 클릭하고, 반려 사유를 입력합니다

**Step 5: 처리 완료 확인**
- 처리가 완료되면 신청자에게 자동으로 알림이 발송됩니다
- "처리 완료" 탭에서 처리 내역을 확인할 수 있습니다

### 인사 담당자: 대체 휴가 등록하기

**Step 1: 대체 휴가 관리 메뉴 접근**
- "인사관리 > 연차설정 > 대체 휴가 관리" 메뉴로 이동합니다

**Step 2: 대상 직원 선택**
- 휴일 근무를 한 직원을 검색하거나 목록에서 선택합니다
- 여러 명을 동시에 선택할 수 있습니다

**Step 3: 대체 휴가 정보 입력**
- 휴일 근무 일자를 선택합니다
- 근무 시간을 입력합니다 (예: 8시간)
- 시스템이 자동으로 1.5배를 계산하여 표시합니다 (예: 12시간 = 1.5일)

**Step 4: 등록 및 확인**
- "등록" 버튼을 클릭하여 대체 휴가를 생성합니다
- 대상 직원에게 대체 휴가 발생 알림이 자동 발송됩니다

**Step 5: 등록 내역 확인**
- "대체 휴가 내역" 탭에서 등록 내역을 확인할 수 있습니다
- 사용 현황 및 소멸 예정 내역도 함께 조회됩니다

## 7. 주의사항/제한

### 정책 제약

#### 연차 사용 단위
- 최소 사용 단위는 0.5일(반일)입니다
- 시간 단위 사용 시에도 0.5일 단위로 반올림 처리됩니다
- 예: 3시간 사용 신청 시 → 0.5일(4시간)로 처리

#### 연차 소멸
- 발생일로부터 2년이 경과하면 자동으로 소멸됩니다
- 소멸 알림을 받았더라도 사용하지 않으면 복구가 불가능합니다
- 회사의 귀책 사유로 사용하지 못한 경우에만 예외적으로 연장될 수 있습니다

#### 대체 휴가 제한
- 발생일로부터 3개월 이내에 사용해야 합니다
- 3개월 경과 시 자동 소멸되며, 금전 보상으로 전환되지 않습니다
- 연차와 대체 휴가를 동시에 신청할 경우, 대체 휴가가 우선 사용됩니다

#### 신청 취소
- 승인 전에는 직접 취소가 가능합니다
- 승인 후 취소는 관리자 승인이 필요합니다
- 연차 사용일 당일에는 취소가 제한될 수 있습니다

### 데이터 반영 시점

#### 연차 생성
- 입사 첫 해: 매월 1일 오전 9시에 일괄 생성
- 입사 2년차 이후: 입사 기념일 오전 9시에 일괄 생성
- 시스템 점검 등으로 지연될 수 있으며, 지연 시에도 소급 적용됩니다

#### 연차 차감
- 승인 완료 시점에 즉시 차감됩니다
- 반려 또는 취소 시 차감된 연차는 즉시 복원됩니다

#### 연차 소멸
- 매년 1월 1일 오전 9시에 일괄 처리됩니다
- 소멸 처리 전까지는 사용 가능합니다

### 자주 발생하는 오해

#### "연차가 자동으로 생성되지 않았어요"
- 입사일 정보가 정확히 등록되어 있는지 확인하세요
- 입사 첫 달에는 연차가 생성되지 않습니다 (익월 1일부터 생성)
- 시스템 생성 시간(오전 9시) 이전에 조회하면 표시되지 않을 수 있습니다

#### "소멸 알림을 받았는데 아직 연차가 남아있어요"
- 알림은 소멸 예정을 미리 안내하는 것으로, 즉시 소멸되는 것은 아닙니다
- 알림을 받은 후에도 소멸일까지는 사용 가능합니다

#### "대체 휴가보다 연차를 먼저 쓰고 싶어요"
- 정책상 대체 휴가가 우선 사용되도록 설정되어 있습니다
- 이는 대체 휴가의 3개월 소멸 기한을 고려한 것으로, 변경이 어렵습니다

#### "퇴사 시 남은 연차는 어떻게 되나요?"
- 퇴사일 기준으로 미사용 연차는 금전으로 보상됩니다
- 인사 담당자가 자동으로 계산하여 최종 급여에 포함시킵니다

## 8. FAQ (자주 묻는 질문)

### Q1. 입사 첫 해에 연차가 몇 일 발생하나요?
**A.** 입사 첫 해에는 매월 1일씩 연차가 발생하며, 최대 11일까지 받을 수 있습니다. 예를 들어 3월 15일에 입사하신 경우, 4월 1일부터 매월 1일씩 발생하여 다음 해 2월까지 총 11일의 연차를 받게 됩니다. 3월(입사월)에는 연차가 발생하지 않습니다.

### Q2. 작년에 사용하지 못한 연차는 어떻게 되나요?
**A.** 미사용 연차는 자동으로 다음 해로 이월됩니다. 다만, 발생일로부터 2년이 경과하면 자동으로 소멸됩니다. 소멸 90일, 60일, 30일, 7일 전에 알림을 보내드리므로, 알림을 확인하시고 소멸 전에 사용하시기 바랍니다. 회사의 귀책 사유로 사용하지 못한 경우에는 인사 담당자에게 문의하시면 예외 처리가 가능할 수 있습니다.

### Q3. 반차는 어떻게 사용하나요?
**A.** 연차 신청 시 사용 단위를 "반일(4시간)"로 선택하시면 됩니다. 반차는 0.5일로 계산되며, 오전 반차(09:00~13:00) 또는 오후 반차(14:00~18:00)로 사용 가능합니다. 시간 단위로 사용하시는 경우에도 최소 단위가 0.5일이므로, 3시간을 신청하셔도 0.5일(4시간)로 처리됩니다.

### Q4. 대체 휴가는 언제 발생하나요?
**A.** 대체 휴가는 법정 공휴일 또는 주휴일에 근무하신 경우 발생합니다. 근무 시간의 1.5배로 계산되어 자동 생성됩니다. 예를 들어 공휴일에 8시간 근무하신 경우, 12시간(1.5일)의 대체 휴가가 발생합니다. 대체 휴가는 발생일로부터 3개월 이내에 사용해야 하며, 기한이 지나면 자동으로 소멸됩니다.

### Q5. 연차 신청을 취소하고 싶어요.
**A.** 승인 전에는 "내 연차 > 신청 내역"에서 직접 취소하실 수 있습니다. 승인 후에는 관리자의 승인이 필요하므로, 관리자에게 취소 요청을 하셔야 합니다. 연차 사용일 당일에는 취소가 어려울 수 있으니, 가급적 사전에 취소해 주시기 바랍니다.

### Q6. 시간 단위로 연차를 사용할 수 있나요?
**A.** 네, 시간 단위 사용이 가능합니다. 다만, 최소 사용 단위가 0.5일(4시간)이므로, 4시간 미만을 신청하셔도 0.5일로 처리됩니다. 예를 들어 2시간 또는 3시간을 신청하셔도 0.5일(4시간)이 차감됩니다. 8시간 미만의 사용은 모두 0.5일 단위로 반올림 처리됩니다.

### Q7. 연차가 자동으로 이월되나요?
**A.** 네, 별도의 신청 없이 12월 31일 기준으로 미사용 연차가 자동으로 다음 해로 이월됩니다. 다만, 이월된 연차도 발생일로부터 2년이 경과하면 소멸되므로, "내 연차 > 연차 현황"에서 소멸 예정일을 확인하시고 계획적으로 사용하시기 바랍니다. 소멸 임박 순으로 자동 사용되므로, 오래된 연차부터 우선 차감됩니다.

### Q8. 퇴사 시 남은 연차는 어떻게 처리되나요?
**A.** 퇴사일 기준으로 미사용 연차는 근로기준법에 따라 금전으로 보상됩니다. 1일당 통상임금을 기준으로 계산되며, 최종 급여에 자동으로 포함되어 지급됩니다. 별도의 신청이 필요하지 않으며, 인사 담당자가 퇴사 처리 시 자동으로 계산합니다. 정확한 금액은 퇴사 전에 인사 담당자를 통해 확인하실 수 있습니다.
